Installation
=============
Installation can be done in two different ways

By Using pip
-----------------

To use Lumache, first install it using pip:

.. code-block:: console

   (.venv) $ pip install lumache


By Using conda
----------------

To use Lumache, first install it using pip:

.. code-block:: console

   (.venv) $ pip install lumache

Check out the :doc:`how-to-use` section for further information.
